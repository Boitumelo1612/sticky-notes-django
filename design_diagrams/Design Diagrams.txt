3️⃣ Sequence Diagram — Create Note Example

Flow: User clicks “Create Note” → fills form → form submits → note saved → list updated

@startuml
actor User
participant Browser
participant Views
participant NoteForm
participant NoteModel
database Database

User -> Browser: Click "Create Note"
Browser -> Views: GET /notes/new/
Views -> Browser: Return NoteForm
User -> Browser: Submit form (POST)
Browser -> Views: POST /notes/new/ with data
Views -> NoteForm: Validate data
NoteForm -> NoteModel: Note.objects.create(...)
NoteModel -> Database: INSERT new note
Views -> Browser: Redirect to notes list
Browser -> User: Display updated notes
@enduml
